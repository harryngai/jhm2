什麼是程式設計 (Programming)?

程式設計 (Programming) 是通過編寫程式碼來解決問題或執行特定任務的過程。

它包括設計邏輯，並使用程式語言 (Programming Languages) 編寫指令，計算機會根據這些指令來執行操作。

程式設計的目標是讓計算機自動完成重複性或複雜的任務。

什麼是演算法 (Algorithm)?

演算法 (Algorithm) 是解決問題的系統化步驟或規則集。

它可以用來處理數據、解決數學問題、或控制程式的流程。

一個好的演算法應該具有正確性 (Correctness) 和效率 (Efficiency)。

程式設計與演算法的關聯

程式設計是將演算法轉化為程式碼的過程。

演算法只是理論上的步驟，而程式則是實際運行的程式碼。

程式設計和演算法是相輔相成的。

什麼是 Python?

Python 是一種高階、直譯式程式語言。由 Guido van Rossum 在 1991 年創建。

它以易讀性、多功能性和跨平台性著稱，是初學者和專業開發者的首選。

為什麼學習 Python?

  • 易於學習: 語法簡單，類似於英文。
  • 龐大的社群: 擁有豐富的支持和資源。
  • 庫和框架: 適用於網頁開發、數據科學、自動化等。

Python 基本語法 (範例)

# 這是一個註解
print("Hello, World!")

name = "Alice"
age = 30
print(f"姓名: {name}, 年齡: {age}")

if age > 18:
  print("你是成年人。")
else:
  print("你是未成年人。")

什麼是 GitHub?

GitHub 是一個基於 Git 的版本控制與協作平台。

它允許開發者託管程式碼倉庫，進行協作開發，並跟蹤程式碼變更。

GitHub 基本工作流程

  • 創建倉庫 (Repository)
  • 克隆倉庫 (Clone the Repository)
  • 進行更改 (Make Changes)
  • 提交更改 (Commit the Changes)
  • 推送更改 (Push the Changes)
  • 拉取請求 (Pull Requests)

什麼是 VS Code?

Visual Studio Code (VS Code) 是由微軟開發的免費、開源的程式碼編輯器。

它以輕量但功能強大、擴展性好、跨平台等特性著稱。

在 VS Code 中設置 Python

  • 從 code.visualstudio.com 安裝 VS Code。
  • 安裝 Python 擴展 (Python Extension)。
  • 配置 Python 解釋器 (Interpreter)。
  • 創建 Python 文件並開始編碼！

程式設計是解決問題並控制計算機行為的核心技術。

Python 是多功能且易於入門的程式語言。GitHub 和 VS Code 是現代開發的必備工具。
